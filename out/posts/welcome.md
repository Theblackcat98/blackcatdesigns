# Welcome to My Blog!

This is your first blog post. It's written in Markdown and will be automatically rendered into a beautiful HTML page.

## What You Can Do

- Write blog posts in simple Markdown format
- Add frontmatter metadata (title, date, tags, etc.)
- Automatically generate a blog listing page
- Create individual post pages with styling

## Getting Started

To add more blog posts:

1. Create a new `.md` file in the `public/posts/` directory
2. Add frontmatter at the top with metadata
3. Write your content in Markdown
4. The post will automatically appear on your blog!

## Markdown Support

You can use all standard Markdown features:

- **Bold text**
- *Italic text*
- [Links](https://example.com)
- ==Highlighting==
- ~~Strikethrough~~ 
- Code blocks with syntax highlighting

```javascript
function hello() {
  console.log("Hello, World!");
}
```

Lists are supported too:

1. First item
2. Second item
3. Third item
